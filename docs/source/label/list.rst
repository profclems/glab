glab label list
---------------

List labels in repository

Synopsis
~~~~~~~~


List labels in repository

::

  glab label list [flags]

Examples
~~~~~~~~

::

  $ glab label list
  $ glab label ls
  $ glab label list -R owner/repository
  

Options
~~~~~~~

::

  -p, --page int       Page number (default 1)
  -P, --per-page int   Number of items to list per page (default 30)
