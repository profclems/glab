glab config init
----------------

Shows a prompt to set basic glab configuration

Synopsis
~~~~~~~~


Update the configuration by setting a key to a value.
Examples:
  $ glab config init
  ? Enter default Gitlab Host (Current Value: https://gitlab.com): |


::

  glab config init [flags]
