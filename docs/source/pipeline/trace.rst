glab pipeline ci trace
----------------------

Work with GitLab CI pipelines and jobs

Synopsis
~~~~~~~~


Work with GitLab CI pipelines and jobs

::

  glab pipeline ci trace <job-id> [flags]

Examples
~~~~~~~~

::

  $ glab pipeline ci trace
  

Options
~~~~~~~

::

  -b, --branch string   Check pipeline status for a branch. (Default is the current branch)
