glab pipeline run
-----------------

Create a new pipeline run

Synopsis
~~~~~~~~


Create a new pipeline run

::

  glab pipeline run [flags]

Examples
~~~~~~~~

::

  $ glab pipeline run
  $ glab pipeline run -b trunk
