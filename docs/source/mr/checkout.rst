glab mr checkout
----------------

Checkout to an open merge request

Synopsis
~~~~~~~~


Checkout to an open merge request

::

  glab mr checkout <mr-id> [flags]

Options
~~~~~~~

::

  -b, --branch string   checkout merge request with <branch> name
  -t, --track           set checked out branch to track remote branch, adds remote if needed
